# PropFind AI

AI-powered Saudi real estate recommendation agent.

## Features
- Natural language property search
- Criteria extraction using GPT-4o-mini
- Follow-up questions for missing information
- Property matching and ranking
- Streamlit interface

## Setup

1. Install requirements

pip install -r requirements.txt

2. Create .env file

OPENAI_API_KEY=your_api_key

3. Run

streamlit run app.py
